# 按钮

## CSS 使用

你可以通过组合`class`来组合出自己想要的组件效果

<demo src="./CSS.vue" />

| 属性           |      值      |
| -------------- | :----------: |
| --color        |   字体颜色   |
| --border-color |   边框颜色   |
| --bg           |   背景颜色   |
| --glitch-text  |   故障文字   |
| --glitch-color |   故障颜色   |
| --top          | 距离顶部距离 |
| --left         | 距离左边距离 |
